Billoo App

I designed UI and developed android app for our final year project called BILLOO @ EPITA. The BILLOO is a Cloud based INVOICE STORAGE PLATFORM. It will enable retailer’s Point of Sale systems to send the receipt/bill digitally from any pay point to the customer’s account through their unique barcode/QR Code/ID without exposing their personal data like mobile number or e-mail address. Due to time contraints, I used modular arch. and was Network only app. with use of SharedPreferences as session management. Retrofit for network calls.

Github App

This is a app that gets the github repositories by keyword search, and on selection of particular repo, it takes you to another screen that shows all commits and all the contributors. It is and offline first application with the help of realm.io. The application is divided into 2 parts: a view with a search field and a list to display the repositories retrieved from the GitHub API, a view with a second list to display branch and contributors with their profile image.
